Linked List: Insertion & Deletion

What is a Linked List?

A Linked List is a linear data structure where elements are stored in nodes, and each node points to the next node in the sequence. Unlike arrays, linked lists don't store elements in contiguous memory locations.

Types of Insertion

Insertion at Beginning

Time Complexity: O(1)

  1. Create a new node with given data
  2. Make new node's next point to current head
  3. Update head to point to new node

Types of Deletion

Deletion from Beginning

Time Complexity: O(1)

  1. Store head node in temp
  2. Move head to next node
  3. Free temp node memory
