Bubble Sort Pseudocode

procedure bubbleSort(A: list of sortable items)
    n = length(A)
    repeat
        swapped = false
        for i = 1 to n-1 inclusive do
            if A[i-1] > A[i] then
                swap(A[i-1], A[i])
                swapped = true
            end if
        end for
        n = n - 1
    until not swapped
end procedure

Implementation Details

Time Complexity

Best Case: O(n)
Average Case: O(n²)
Worst Case: O(n²)

Space Complexity

Auxiliary Space: O(1)

Key Characteristics

  • Simple implementation
  • Stable sorting algorithm
  • In-place algorithm
